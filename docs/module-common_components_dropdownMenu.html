Module: common/components/dropdownMenu

Displays a dropdown menu from a list of clickable actions

Parameters:
Name Type Description
actions Array.<String> | Array.<Object> = Array of menu items. Actions can be String or Object {text: string, click: function(model,action) }
click optional | function = default function to run when clicking an item. function(model, action)
model optional | any = data to pass to to action functions
align optional | String @ left, center, right (default: inherit / left)
valign optional | String @ vertical position of menu compared to its trigger button. top | default: bottom
position optional | String @ position of menu in container default: fixed (default: absolute)
variable optional | String @ global variable name
trigger optional | String @ event type to open menu: click | hover (default: hover)
triggerHover optional | Boolean =? when true, enables dropdown on hover (default: true)
triggerClick optional | Boolean =? when true, enables dropdown on click (default: false)
openInstance optional | Object =? last open menu instance (to prevent multiple menus open at once). Set to current controller instance when opening menu / null when closing. Other menu instances sharing the same openInstance variable are closed when another opens.
selectedItem Array.<object> =? Last selected item
search optional | Boolean =? if true: display search field / default: false
searchText optional | String =? search text field value
