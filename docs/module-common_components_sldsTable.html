Module: common/components/sldsTable

Displays an array of objects in an HTML table. Every column can be made read-only or editable using different field types. Component handles instant sorting by any column on the client side.

Parameters:
Name Type Description

[{ field: "$selected" , type: "checkbox" , label: "is selected", ... }, ... ]
