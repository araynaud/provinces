Used for displaying Lookup(search)/Select/Multi-select input fields

Name Type Description

multiple optional | Boolean =? if true: multiple select / default: single select
readOnly optional | Boolean =? true: read-only / default: editable
variable optional | String @ global variable name (for console debugging only)
inputId optional | String @ id of search input field

remoteActionArgs optional | Array < remote action arguments, passed before search term

initAction optional | String @ name of the method to call to reload selected item

onChange optional | function =? function called on adding item (if onAdd not passed) and on removing item (if onRemove not passed)
onAdd optional | function =? function called when an item is selected (if onAdd is not passed, onChange will be executed instead)
onRemove optional | function =? function called when an item is deselected (if onRemove is not passed, onChange will be executed instead)
options optional | Array.<String> | Array.<Object> =? list of options to search / select from (ex: answer options from the question)
displayField optional | String @ name of the item object field to display (default: name)

exclude optional | Array.<Object> =? array of objects to prevent selection / remove from search results (diff by modelField or keyField)

minLength optional | Number =? minimum search term length before searching (default: 2 for search / 0 for multi-picklist)
icon optional | String | function @ item icon to use. can be a String if it is the same for all items ("account")

itemIcon optional | function =? function(item) function(item) returning icon name and file depending on item properties.
fieldType optional | String @? actual type of field that used to save data 'search'/'multi-picklist'
required optional | String | function < attribute to control required state out of formly-form context
bulkSelection optional | Boolean =? for multiple selection: enable select all options / none
scrollableContainerToWatch optional | String @ (jquery selector) if container is scrollable, must know an attribute to detect when it scrolls
maxLength optional | Number =? max length for picklist answers stored as string,
watchModel optional | Boolean =? enable refresh on vm.model change (if value can be passed from outside after initalizing)
watchOptions optional | Boolean =? enable refresh on vm.options change (if options can be passed from outside after initalizing)
