Module: common/components/sldsNotification

This component can be used to display single or multiple notification messages. Each notification contains a message and due date, and can be closed manually or after a specified duration.
Parameters:
Name Type Description
notifications Array.<Object> = notification objects to display
visible Boolean = controls visibility of the component. true: open / default: closed
variable optional | String @ global variable name
message optional | string @
duration optional | Number @
html optional | Boolean =
icon optional | string @
