Module: common/components/sldsTabs

Displays a horizontal tab bar from an array of strings or objects

Parameters:
Name Type Description
model string = the selected tab value
tabs Array.<object> =? list of tabs to display [ { title: "Tab title" }, ... ]
tabTitles Array.<string> =? list of tab titles to display
defaultTab number | string | object =? tab title or tab object
selectedTabIndex number =? index of selected tab
onClick function =? default function when clicking on a tab
Source:
  • common/app/components/sldsTabs/sldsTabs.js, line 1
See:
  • Tabs - Lightning Design System
Example
//with array of strings
vm.tabs = ["TABLE", "JSON", "TREE"];
...
<slds-tabs model="vm.selectedTab" tab-titles="vm.tabs"></slds-tabs>

<div ng-if="vm.selectedTab == 'TABLE'">content of tab 1</div>
<div ng-if="vm.selectedTab == 'JSON'">content of tab 2</div>
<div ng-if="vm.selectedTab == 'TREE'">content of tab 3</div>
