Module: common/components/formComment

This module lets you to interact over an individual field in a form within a workflow. You can add comment for a question/field inside the form if you have some input/feedback on that field and send it in a request.
Parameters:
Name Type Description
comments Array.<Object> = List of the comments associated to the current question
visible Boolean = true: popover is visible / otherwise hidden
variable optional | String @ global variable name
questionId String = the id of the associated question
containerQuestionId String =? the id of the associated question (for product test comments)
readOnly optional | Boolean = makes the editable fields in the popover readonly
nubbinPosition String = position of the pointing arrow for each popover(which direction it should point)
element optional | HTMLElement = the HTML element for this component

Example
<form-comment ng-if="to.comments" visible="to.showCommentBox" comments="to.comments" question-id="options.key" read-only="$root.isRequestClosed" nubbin-position="to.commentNubbinPosition"></form-comment>
